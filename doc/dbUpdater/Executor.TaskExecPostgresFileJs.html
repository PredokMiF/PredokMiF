Class: TaskExecPostgresFileJs

Executor. TaskExecPostgresFileJs

new TaskExecPostgresFileJs(config) → {Executor.TaskExecPostgresFileJs}

Выполняет JS файлы. Имена файлов должны соовпадать с регуляркой /^.*\.postgres\-file\-js\.js$/ т.е. файл должен заканчиваться на ".postgres-file-js.js" внутри должен быть валидный NodeJS код. Можно использовать require. Сама задача должна быть оформлена в виде модуля и возвращать функцию с интерфейсом (dbClient, cb)
